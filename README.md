# authorarchive

A LaTeX style for producing author self-archiving copies of (academic) 
papers. 

## Installation 
Copy `authorarchive.sty` in a directory that is searched by LaTeX
(e.g,. either your `texmf` tree or the local directory with your main
LaTeX file.
